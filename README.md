# flex-talk
reading discord bot

しかし、既存の物は読み上げ回数に制限があり複数のBOTをサーバーにJOINさせても尚リソースが足りないなんてことが日常茶飯事・・・

そこでもういっそ自分で作っちまえばええやんと３時間くらいでぱぱっと適当に作りました。
ちなみにこれは別で最終形態が存在しておりそれの本当に最初のベースとなったプログラムになります。
